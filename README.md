# Solavisetech-women-in-data-cohort2
# About The Program
The solaviseTech women in data cohort2 training program is a program designed for women.It is focused on empowering women with data and technology skills through practical learning and collaborations.
# About Me
My name is Appiah Gloria Yeboah,a Ghanaian Information technology graduate with interests in UI/UX design,backend development and data science and analysis  who is also an entrepreneur.
# My Expectations
My expectation from this program is to gain practical knowledge,improve my technical skills,work on real-world project and make connections with other learners and mentors.
# What I Hope To Learn
During this training I hope to learn what data science and analysis is about,learn the python language and how to use it to solve data related issues, learn data cleaning and management issues and also get a better understanding of Git and Github.
